# Ayurveda-Hotel-Management-System

# overview
this is a full-stack comprehensive web-based management system for Sath Villa Ayurveda, designed to streamline both hotel operations and customer interactions. The system allows users to easily book Ayurveda packages, purchase Ayurveda products, and submit inquiries and reviews through a smooth and user-friendly interface.

For the admin side, the platform includes powerful management features such as package management, product management, booking management, user handling, and review moderation. This ensures that all hotel operations can be handled efficiently in one place.

Overall, the system enhances the guest experience while simplifying daily tasks for the hotel staff, making the entire process more organized and efficient.

# Technologies Used
- Frontend:
  - React.js
  - HTML
  - CSS
  - JavaScript
  - Axios (for API calls)

- Backend:
  - Node.js
  - Express.js
  - MongoDB
 
  # Features
  1. User Registration and Login:
     - Allows users to create an account and securely log in to access personalized services.
  2. Book Ayurveda Packages
     - Enables users to select and book treatment packages online with date and time selection.
  3. Purchase Ayurveda Products
     - Lets users browse and buy Ayurveda products directly from the website.
  4. View Packages & Products
     - Displays details, pricing, and descriptions of available packages and products.
  5. Submit Inquiries
     - Allows users to ask questions or request information directly from the hotel.
  6. Write & View Reviews
     - Users can provide feedback on their experience and read reviews from others.

 # Admin Features
  1. Package Management (CRUD):
     - Admin can create, read, update, or delete Ayurveda treatment packages.
  2. Product Management (CRUD):
     - Admin can manage Ayurveda products including adding new items or updating inventory.
  3. Booking Management
     - Admin can view, approve, or cancel bookings made by users.
  4. User Management
     - Admin can manage user accounts, roles, and access privileges.
  5. Inquiry Management
     - Admin can view and respond to customer inquiries efficiently.
  6. Review Moderation
     - Admin can monitor and manage user reviews to maintain quality feedback.
